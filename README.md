# hw-css

## install

```
$ bower install healthwatch/hw-css --save
```

## usage

```html
<link rel="stylesheet" href="bower_components/hw-css/css/hw.min.css"> <!-- for WWW -->
<link rel="stylesheet" href="bower_components/hw-css/css/grp.min.css"> <!-- for GRP -->
```
